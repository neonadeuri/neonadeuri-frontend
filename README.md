## Start

터미널에서 아래먼저 쳐서 local 환경을 맞추고

```
  yarn setup
```

local server 올림

```
  yarn dev
```

Tailwind를 동적으로 실시간 보면서 코딩하고싶다 하면 아래

```
  yarn dev:tailwind
```

Storybook 사용

```
  yarn storybook
```

나머지는 package.json의 스크립트 참고하세요
