# 달밤 프론트에 오신걸 환영합니다.

**포트폴리오를 위한 프로젝트입니다.**

## 시작단계

### install

```
  yarn setup
```

### Start

**설치물 extentions**
prettier
ZipFS
Tailwind CSS IntelliSense
ESLint

![extentions](./utils/installextentions.png)

```
  yarn dev
```

Tailwind를 동적으로 실시간 보면서 코딩하고싶다 하면 아래

```
  yarn dev:tailwind
```

나머지는 package.json의 스크립트 참고하세요
